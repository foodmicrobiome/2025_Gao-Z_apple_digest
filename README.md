# 2025_Gao-Z_apple_digest

Microbial communities were recovered from apples using culture enrichments, then supplemented into into different food products. Survival of the natural and enriched microbiomes in the food matrices were evaluated under simulated gastric digestion.

**Reference:** in review

# Contents

**code**: bioinformatics scripts and R code used in the analysis

**sample_data/metadata**: tables with sample information related to apple microbiome source (cultivar), sample type (apply, enrichment, digesta), food matrix (applesauce, DI water, sweet potato puree, chicken puree, water-in-oil emulsion) 

**sample_data/taxa_tables**: microbiota taxa counts tables and intermediate/output files in the analysis
